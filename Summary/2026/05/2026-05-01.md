## 🎓 今日の基礎講座：ステーブルコインって何？

今日のメタ・ビザ・SBIの3件をすべて貫く主役が**「ステーブルコイン」**。ここを理解すれば、3行まとめの本当の意味が見える。

**ステーブルコインとは:**
**「米ドル等の法定通貨に1:1で価値を固定した仮想通貨」**のこと。代表例は**USDC**（米サークル社発行、約700億ドル流通）、**USDT**（テザー社発行、約1,500億ドル流通）。1USDC = 1ドル、1USDT = 1ドルで取引される。

**普通の仮想通貨との違い:**
- ビットコイン → 価格が秒単位で変動、決済には不向き
- ステーブルコイン → 価格が1ドル固定、決済・送金に最適

**裏付けの仕組み:**
発行体（サークル・テザー）が**現金と短期米国債で100%の準備金**を保有。1USDC発行する=1ドルの米国債を金庫に積む。米GENIUS法（2025年成立）で「準備金は安全資産のみ」と義務化された（前回04-28モルガン・スタンレーMMF記事の背景）。

**なぜ今、巨大企業が一斉に採用するか:**
①世界どこでも数秒・低手数料で送れる、②24時間365日動く、③スマートコントラクト（自動決済プログラム）と連携できる、④2025年の世界取引総額は**33兆ドル**（米GDPに匹敵）。Visaの取扱が前四半期+50%伸びている速度がこの普及を物語る。

**ひとことで言うと** — Facebook・Instagram運営の米メタ・プラットフォームズが、コンテンツクリエイターへの収益支払いに米ドル連動ステーブルコイン[※1]「USDC[※2]」を採用。**ソラナ（Solana）[※3]とポリゴン（Polygon）[※4]**両ブロックチェーン上で送金、決済処理は米ストライプ（Stripe）が担う。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| 発表企業 | メタ・プラットフォームズ |
| 採用通貨 | USDC（米サークル社発行のドル建てステーブルコイン） |

| 拡大方針 | 段階的に対応国を拡大 |

**過去経緯:**
メタは2019年に独自ステーブルコイン構想「Libra（後にDiemへ改名）」を発表したが、各国中央銀行・規制当局からの強い反発で頓挫し、**2022年に正式撤退**。今回はそのリベンジに見えるが、構造は正反対：**「自社で通貨発行・運用しない、すでに規制対応が進んだUSDCに乗る」**戦略を選択。決済インフラもストライプに委託し、メタが直接ブロックチェーン上の処理を担わない。**規制対応コストを大幅削減**した格好。

**なぜ知っておくべきか**
仮想通貨業界にとって**歴史的な節目**。世界最大級のSNSプラットフォーム（メタの月間アクティブユーザーは約40億人）が決済レイヤーにステーブルコインを採用するということは、**「ステーブルコインがマス層のインフラに到達した」**ことを意味する。新興国（コロンビア・フィリピン）から始める戦略も賢い：①銀行口座普及率が低くウォレットが普及している、②米ドル決済需要が強い、③規制が比較的緩い—の3条件が揃う市場。Solana採用の背景はステーブルコインのデイリーアクティブユーザー数（DAU）[※5]が4/27に過去最高を更新するなど、低手数料・高速決済を活かした成長があります。

**ビジネスへの応用**
中小企業の経営者・人事担当者が押さえるべき視点。①**海外フリーランス・ギグワーカーへの支払い**にステーブルコイン送金が選択肢として現実化。SWIFT送金（数日・数千円の手数料）を、USDC送金（数秒・数十円の手数料）に切り替える流れ。②**自社サービスにクリエイター・インフルエンサー報酬支払いがある場合**、ストライプのUSDC機能を使えば既存のStripeアカウントから即時切替可能（日本も今年中に対応見込み）。③メタが先導する**「自社で通貨発行せず、既存ステーブルコインに乗る」**モデルは、楽天・PayPay等の国内大手が今後検討する標準的な形。自社がそのインフラ提供側に回る機会も視野に入れる。

### ★★★ SBI証券・大和証券らDCJPYでデジタル証券DVP決済 国内初の検証完了
**ソース:** [SBI証券・大和証券ら6社、「DCJPY」活用のデジタル証券DVP決済の検証完了、国内初](https://www.neweconomy.jp/posts/569492)（あたらしい経済）

**ひとことで言うと** — SBI証券・大和証券・SBI新生銀行・ブーストリー・大阪デジタルエクスチェンジ（ODX）・ディーカレットDCPの**6社**が、トークン化預金[※9]「DCJPY[※10]」を活用したセキュリティトークン[※11]（ST/デジタル証券）の**DVP決済（証券と現金の同時交換）[※12]**の検証完了を4/24に発表。STとデジタル通貨を実発行してDVP決済を検証した事例は**国内初**。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| 発表日 | 2026年4月24日 |
| 参画6社 | SBI証券、大和証券、SBI新生銀行、ブーストリー、ODX、ディーカレットDCP |
| 実証対象ST | ディーカレットDCPのデジタル社債（ST社債） |

| ST基盤 | ブーストリーの「ibet for Fin」 |
| 通貨基盤 | SBI新生銀行発行のDCJPY |
| ODX役割 | オブザーバー参加（将来のPTS[※13]決済での利用見据え） |

**従来課題と解決策:**
これまで国内ST市場では、**ブロックチェーン上のST受渡しは即時**である一方、**資金決済は銀行振込**で行われており、決済タイミングの不一致が**決済リスク**（一方が破綻すると相手方が取りはぐれる）と**事務負担**を生んでいた。今回の実証では、ibet for FinとDCJPYネットワークを連携させ、**ST送付と日本円決済が一体で実行される**ことを確認。決済不履行が起きえない構造になる。

**なぜ知っておくべきか**
**日本のデジタル証券インフラが「実用フェーズ」に到達**した象徴的な事例。①国内大手証券2社（SBI・大和）が直接参加することで、**業界標準としての位置付け**が固まる。②今回検証されたDCJPY-ST連携は**「デジタル日本円×デジタル証券」**の組み合わせで、米国でモルガン・スタンレーやBlackRockがやっている「USDC×トークン化米国債」の日本版インフラに相当。③**今後はこの仕組みの上に不動産STO（小口分散投資）、社債STO、株式STO**が積み上がる。証券会社の決済オペレーションコストが大幅圧縮され、**個人投資家でも100万円以下の小口社債・小口不動産投資**が標準化する2027〜2028年が現実視野に。

**ビジネスへの応用**
中小企業の財務担当・経営者が押さえる3点。①**自社の社債発行コストが下がる可能性**。従来の社債発行は数千万円のリーガル・印刷コストがネックだったが、ST化で1/10以下になる試算（みずほ・SBI試算）。中小企業の数億円〜数十億円規模の資金調達手段として現実化。②**不動産・債権・知財の小口流動化**。本社ビル・自社物件・売掛債権をST化して投資家から資金調達する選択肢が拡大。③**海外事業者との取引**でも、DCJPY建てのST活用で為替リスクを排除した社債発行が可能に（中長期）。日本企業が「グローバル投資家から円建てで直接調達」できる新しい資金調達ルート。

**DeFi史上最大級の自助復旧事例**になりつつある。①前回04-28で扱った「**Curve創業者の業界自治要請**」が現実に機能した形。規制当局が動く前に業界が自浄して事件を解決。②**LayerZero、Lido、Mantle、Consensus、Circleなど業界の主要プレイヤーがすべて拠出**しているのは、「DeFiは助け合いコミュニティ」という理念がただの建前ではなく、137,000ETH（約3.3億ドル/約500億円）を実費で動かせる実態があることの証明。③ただし**ガバナンス承認のプロセスが残る**ため、すべての約束が予定通り執行されるかは流動的。④この方式が成功すれば、今後のDeFi被害事案で「**業界連合による復旧スキーム**」が標準テンプレートになる。

**ビジネスへの応用**
中小企業の財務担当・IT責任者が押さえる3点。①**DeFiプロトコル選定の評価軸が変わる**。単なるTVL（預け入れ総額）や利回り％だけでなく、**「事故時の業界連帯支援が見込めるか」**が評価軸に。アーベのように業界主要プレイヤーが守る側に回るプロトコルは事実上の「Too big to fail」化。②**自社が仮想通貨を運用する場合**、単一プロトコルへの集中を避け、業界の主要連合に組み込まれているプロトコルに分散するのが現実的。③**保険商品化の波**。Nexus Mutual等のDeFi保険のニーズが急拡大、損保会社の参入機会としても注目すべき。

## マイニング・インフラ

### ★★★ ソラナ財団、量子耐性Falcon署名へ移行計画｜Anza・Jump Crypto研究で本格策定
**ソース:** [ソラナ、量子耐性に向けた移行計画を公表。Falcon署名を有力候補に](https://www.neweconomy.jp/posts/570085)（あたらしい経済）

**ひとことで言うと** — レイヤー1ブロックチェーン[※21]ソラナを支援するソラナ財団が、4/27に**量子コンピューター[※22]への移行計画**を公式に公表。**「Falcon（ファルコン）」と呼ばれるポスト量子デジタル署名方式**を有力候補に位置づけ、ソラナ技術開発のAnzaとファイアダンサー開発のJump Cryptoが独立に研究して同じ結論に到達。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| 発表日 | 2026年4月27日 |
| 発表主体 | ソラナ財団（Solana Foundation） |
| 候補署名方式 | Falcon（ファルコン） |
| 研究主体 | Anza、Jump Crypto |
| 現状認識 | 量子コンピューターの脅威は「数年先」、急ぐ必要はないが事前準備が必要 |
| 既存実装 | Blueshiftの「Winternitz Vault」が2年以上前から稼働 |

**Falconが選ばれた理由:**
ポスト量子署名方式は一般にデータサイズが大きくなる傾向があるが、ソラナのような高速処理ネットワークでは**トランザクションごとのデータ増加が性能に直結**。Falconは**比較的署名サイズが小さい**ため、ソラナの性能要件と両立しやすい。

**移行ロードマップ:**
①研究継続・代替案の評価、②量子コンピューターが現実的脅威となった時点で**新規ウォレットにポスト量子署名を導入**、③段階的に既存ウォレットも移行。

**なぜ知っておくべきか**
**ブロックチェーン全体の存続問題**に対する主要L1の本格対応が始まった。①前回04-28アプトス機密性、04-27スタークネットShinobiと並んで、**主要L1/L2が同時並行で次世代対応を進めている**流れの最新事例。②量子コンピューターは「いつ実用化されるか」がほぼ全業界の不確実要因だが、4/24に量子15ビット鍵解読報道があり、**「数年先」と言われた時計が前倒しに動いている可能性**。③ソラナはMARA財団（同じ4/27設立）も「BIP360」量子耐性提案を支援しており、**業界横断の本格対応モードに入った**。Falcon方式は**ETHにも提案されている**ため、業界デファクトスタンダードに近づきつつある。

**ビジネスへの応用**
中小企業のIT・セキュリティ責任者が押さえる3点。①**量子耐性は仮想通貨に限らず、ITセキュリティ全般の3年後の主戦場**。SSL/TLS、デジタル署名、銀行認証システムまで影響範囲が広い。社内システムでこの対応に着手しているか確認すべき。②**自社が仮想通貨を保有する場合**、ウォレットアドレスは将来的にポスト量子署名対応のウォレットへの移行が必要になる。今のうちから「移行可能な設計のウォレット」を選ぶ。③**ソラナ・アプトス等の主要L1がこの議論で主導権を取りつつある**事実は、**ETHやBTCの相対的価値**にも長期で影響する可能性。BTCはBIP360（MARA財団支援）対応が始まったばかりで、**Falconが業界標準になればソラナ/イーサリアム系が技術優位**を取りうる構造変化に注意。

## 💡 今日の理解度チェック（3問）

**Q1.** ★入門 「ステーブルコイン」と普通のビットコイン・イーサリアムの一番の違いは何か。1点挙げよ。
**Q2.** ★★応用 メタが2019年「Libra」で潰され、2026年に「USDC採用」で再参入した戦略の決定的な違いを2点挙げよ。
**Q3.** ★★★発展 SBI証券・大和証券らDCJPY×ST DVP決済が「国内初」とされる本質的な意義を、米国のステーブルコイン×米国債トークン化エコシステムとの比較で論じよ。

<details>
<summary>答えを見る</summary>

**A1.** **価格安定性**。ビットコイン・イーサリアムは需給で価格が秒単位で変動するが、ステーブルコインは米ドル等の法定通貨に1:1で固定（1USDC = 1ドル）。これにより**「決済・送金・保管の手段」**として機能し、価値変動を恐れずに使える。裏付けは現金と短期米国債で100%担保され、米GENIUS法（2025年）で「準備資産は安全資産のみ」と義務化された。

**A2.** **違い1（通貨発行主体）:** Libraは**メタ自身が独自通貨を発行・運用**する構想で、各国中央銀行から「グローバル貨幣を一企業が支配する」と猛反発を受けて頓挫。今回はサークル社という第三者が既に発行・規制対応済みのUSDCに「乗る」だけで、メタは通貨発行主体ではない。**違い2（インフラ運用）:** Libraはメタが独自ブロックチェーンとウォレットを構築する想定で、規制対応が膨大だった。今回は決済処理を米ストライプに委託、ブロックチェーンも既存のSolana/Polygonを使うため、メタが直接規制対応するレイヤーがほぼない。**示唆:** 「自前でやらない」割り切りが規制突破の鍵。今後の大手プラットフォーム（楽天、PayPay、Microsoft、Amazon等）が仮想通貨に再参入する際の標準的なテンプレートになる。

**A3.** **米国モデル:** USDC（民間ステーブルコイン）×トークン化米国債（BlackRock BUIDL等）×取引基盤（Securitize等）の組み合わせで、**民間主導でドル建てデジタル金融市場が拡大**中。米GENIUS法は事後的に枠組みを与えた。**日本モデルの本質:** DCJPY（民間銀行発行のトークン化預金）×ST（民間証券基盤ibet for Fin）×大手証券会社（SBI・大和）の組み合わせで、**「日本円建てデジタル証券市場」**の業界標準を構築。これは①日本円の電子化（中央銀行デジタル通貨CBDCではなく、民間銀行のトークン化預金経由）、②証券会社の決済オペレーション効率化、③小口社債STO・小口不動産STO等の**新しい資金調達ルートの実装基盤**になる。**米国との違い:** 米国はUSDC（ドル建て）が主役で米国債需要拡大を制度化したのに対し、日本はDCJPY（円建て）が主役でドル経済圏との接続をどう設計するかが今後の課題。**結論:** DCJPY×ST DVPは技術検証ではなく、**「日本のデジタル金融市場のOS」**の業界合意形成プロセスであり、米国モデルへの「日本独自の対抗ロードマップ」を意味する。今後の不動産・社債・株式STOの全てがこの基盤の上に積み上がる。
