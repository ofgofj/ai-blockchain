## 🎓 今日の基礎講座：ETF（上場投資信託）って何？

今日の記事に**仮想通貨ETF**の話が複数出てくる（カントンコインETFのナスダック上場、JPX2027年ETF構想、ストラテジー優先株連動ETPなど）。ここがわからないと、なぜ「ETF上場」がそんなに重要視されるかが見えない。

**ETF（Exchange Traded Fund、イーティーエフ＝上場投資信託）**とは、株や債券などの「複数の資産をまとめた箱（ファンド）」を、株式と同じように証券取引所で売買できる金融商品のこと。

**仮想通貨ETFの場合の仕組み:**
1. 21シェアーズなどの運用会社がビットコインやカントンコインを実際に買って保管する
2. その保有を裏付けに「ETFの株（受益証券）」を発行し、ナスダックなど証券取引所に上場する
3. 投資家は普通の証券口座から、そのETFを株のように売買する
4. 投資家自身は仮想通貨を保有しないが、ETFの価格は仮想通貨価格に連動する

**何が便利なのか:**
- **証券口座だけで完結**：仮想通貨取引所の新規開設、ウォレット[※7]管理、秘密鍵[※8]の保管リスクが不要
- **税制が株式と同じ**：日本の場合、暗号資産は「雑所得（最大税率55%）」だが、ETFは「申告分離課税（20.315%）」が見込まれる
- **証券会社の通常業務に組み込める**：IFA[※9]・銀行窓口・iDeCo・NISAなど既存の販売チャネルが使える

**なぜ「現物ETF」が画期的か:**
ETFには「先物ETF」（先物商品で価格連動を作る）と「現物ETF」（実際に資産を保管する）がある。**現物ETFは需要が増えるたびに運用会社が現物を買い増す必要があるため、価格に直接インパクトが出る**。2024年1月の米ビットコイン現物ETF承認以降、機関投資家の資金が一気に流れ込んだのはこの理由。

今日の記事では、このETFに関するニュースが**3件以上**あります（カントンコイン現物ETF上場、ストラテジー優先株連動ETN、JPXの暗号資産ETF構想）。

## 決済・送金

### ★★★ 3メガバンク・ブラックロック連合、日本国債のトークン化と「T+0レポ取引」を5月から共同検討
**ソース:** [プログマら、「トークン化国債」WG設置。日本国債のトークン化とT+0レポ取引を検討](https://www.neweconomy.jp/posts/572117)（あたらしい経済）／[日本国債をブロックチェーン上で取引、メガバンクらが共同検討](https://crypto-times.jp/news-japan-jgb-tokenization-onchain-repo-progmat/)（CRYPTO TIMES）

**ひとことで言うと** — 日本国債を「ブロックチェーン上で動かせるトークン」に変えて、その日のうちに担保取引（レポ取引[※10]）を完結させる仕組みを、3メガバンク＋ブラックロック＋大和証券など金融機関30社近くで検討する作業部会が動き出した。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| 主催 | プログマ社主催「デジタルアセット共創コンソーシアム（DCC）」 |
| WG名称 | トークン化国債・オンチェーンレポ ワーキング・グループ |
| 検討対象 | 日本国債を「Tokenized JGB（TJGB）」としてトークン化、ステーブルコイン[※11]で貸借 |
| 主要参加 | 3メガバンク・信託銀行、ブラックロック・ジャパン、東京海上HD、大和証券、SBI証券、日本証券金融、Ava Labs（アバランチ[※12]）、Digital Asset（カントン[※13]） |
| キックオフ | 2026年5月（約3週間ごとに検討会） |
| 報告書公表 | 2026年10月（税制・法的論点を整理） |
| 商用化目標 | 2026年内にTJGB組成プロジェクト開始 |

**「T+0レポ取引」とは何か:**
レポ取引は国債を担保にお金を短期で借りる取引で、世界全体で約16兆ドル（約2,509兆円）の規模を持つ。現在の日本のレポ決済は**「T+1」（取引成立翌日）が標準**。これを**「T+0」（当日中に建てて当日中に解消）**にすることで、銀行は「日締め時点でバランスシート[※14]に取引が残らない」状態を作れる。これは**自己資本比率規制・レバレッジ規制[※15]の負担軽減**に直結する大きな技術改革。

**なぜ知っておくべきか**
①日本国債は世界トップクラスの巨額市場（日本のレポ市場は世界の約1割）であり、ここがオンチェーン化すると**「アジア時間帯の機関投資家マネー」がブロックチェーン上に来る**、②前回扱った米国のDTCCトークン化（10月開始）・オンド連合のRWA[※16]実証と**地理的に補完関係**で、米国・日本が並んで動き出した構図、③参加企業の顔ぶれが「3メガ＋ブラックロック＋ブロックチェーン基盤大手3社」と完璧。**規制当局の事前合意なしには集まれない布陣**で、政策的なゴーサインが出ていることを暗示。

**ビジネスへの応用**
①**中小企業の余剰資金運用先**として、いずれ「日本国債担保のオンチェーン短期運用商品」が選択肢に入る可能性が高い。今は機関投資家限定でも、商品設計次第で1〜2年後に法人向け口座で扱われる可能性あり。メインバンクから「TJGB対応の有無」を聞ける段階に来た、②**自社のステーブルコイン受取の準備**は今年中に検討開始しておくとよい。日本の3メガ共同ステーブルコイン構想（前回触れた）とTJGBが組み合わさると、**B2B決済の標準形態が変わる**可能性が大きい、③**自社のIR・財務戦略文書**で「ブロックチェーン技術への対応姿勢」を1段落でも明文化しておくと、来年以降の金融機関・取引先評価で有利に働く。日本経済新聞報道の「デジタル証券発行累計3,600億円」という数字は、まだ大半が不動産STだが**国債STが入ると桁が変わる**規模。

## 企業・機関投資家

### ★★★ 米2大証券モルガン・スタンレー＆シュワブが「証券口座で仮想通貨直接売買」を本格提供開始
**ソース:** [モルガンとシュワブが仮想通貨直接取引に本格参入](https://crypto-times.jp/news-morgan-and-schwab-make-a-full-scale-entry-into-direct-cryptocurrency-trading/)（CRYPTO TIMES）

**ひとことで言うと** — 米国の伝統的な大手証券会社2社が、**普通の証券口座から株を買うのと同じ操作でビットコインを買えるサービス**を始めた。仮想通貨専業の取引所を経由しなくてよい時代に入った。

**何が起きたか**

| 会社 | 体制 |
|---|---|
| モルガン・スタンレー | 傘下E*Tradeで仮想通貨取引、Zerohash経由で執行、2026年上半期に開始予定 |
| チャールズ・シュワブ | 自社「シュワブ・プレミアー・バンク」で保管、Paxos経由で執行、BTC・ETH[※17]から段階提供 |

**この動きを後押しした規制緩和の流れ:**
- **2025年3月: FDICが仮想通貨活動の事前承認要件を撤廃**（銀行が自由に仮想通貨業務を扱えるように）
- **2025年5月: OCCが国法銀行による顧客管理の仮想通貨売買を認める見解を表明**
- **2025年7月: スタンダード・チャータード銀行が機関投資家向けBTC・ETH現物取引を開始**

つまり**「規制当局がGOサイン → 1〜2年で大手銀行・証券が一斉参入」**という典型的パターンが完成した。

**仮想通貨専業との対比:**
ロビンフッドの2026年Q1の仮想通貨名目取引量は**前年比48%減の240億ドル**。コインベースも2四半期連続赤字（同日報道）。**専業モデルの逆風と伝統金融の参入が同時に起きている**。

**なぜ知っておくべきか**
①**「仮想通貨は専業取引所で買うもの」という常識が崩れる**。投資家側から見れば、複数アカウント管理・税務処理の手間・カストディ[※18]リスクが一気に減る、②**ETFと並んで、機関投資家・富裕層が動きやすいパスが2本目に開いた**。ETFは「指数連動の箱を買う」、直接取引は「現物そのものを口座に置く」と用途が違うため、両方が需要を吸い上げる、③日本では**SBI証券・楽天証券が同様のサービス**を展開する可能性があるが、税制（雑所得55%）の壁があるため、米国ほど一気に進まないと予想される。**JPX2027年ETF構想**（前回触れた）が日本側の突破口になる可能性が高い。

**ビジネスへの応用**
①**自社の経理部・財務部の研修内容**に「仮想通貨を証券会社経由で保有する場合の会計処理」を追加する時期。資産計上区分・期末評価・税務処理が今後数年で標準化される、②**自社の取引先（特に米国子会社・取引先）が仮想通貨で決済する要望**を出してくる可能性が一段上がった。「対応できる体制」を社内で議論しておくべき段階、③**自社の投資家向け資料**で、保有資産に「仮想通貨カテゴリ」を新規追加する企業が今年から急増する見込み。同業他社のIRをチェックする習慣をつけたい。

## AI × ブロックチェーン

### ★★★ サークル、AIエージェント向け超少額決済「ナノペイメンツ」をメインネット稼働開始
**ソース:** [サークル、AIエージェント向けUSDC少額決済「ナノペイメンツ」をメインネットで提供開始](https://www.neweconomy.jp/posts/572030)（あたらしい経済）

**ひとことで言うと** — USDCの発行元サークルが、**1回0.000001ドル（約0.00016円）から、しかも手数料なしでUSDCを送れる**仕組みを本番運用開始。AI同士が自動で代金を支払い合う「エージェント経済」の決済インフラが整い始めた。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| サービス名 | Nanopayments（ナノペイメンツ） |
| 開始日 | 2026年4月29日 メインネット移行 |
| 最低送金額 | 0.000001ドル相当のUSDC |
| ガス代[※19] | 個別支払いごとには発生しない（一括処理で吸収） |
| 対応規格 | x402（HTTP 402ベースの決済プロトコル[※20]） |
| 対応チェーン | アービトラム、アバランチ、ベース、イーサリアム、ハイパーEVM、OPメインネット、ポリゴンPoS、セイ、ソニック、ユニチェーン、ワールドチェーンの11チェーン |
| 基盤 | サークル・ゲートウェイ（複数チェーンのUSDC残高を一元利用） |

**仕組みのキモ:**
従来のオンチェーン送金は1回ごとにガス代（イーサリアムだと数百円〜数千円）が必要で、**1円以下の支払いは技術的に不可能**だった。ナノペイメンツは**「支払い承認をオフチェーンでまとめ、後からまとめて一括決済」**という方式で、個別支払いの摩擦をゼロに近づけた。**HTTP 402（インターネット標準の「支払い要求」コード）**を使うため、既存のWebサービスに組み込みやすい。

**ユースケース（AIエージェント経済）:**
- AIエージェントが他のAIエージェントから1ファイル0.001ドルで画像認識サービスを買う
- API呼び出し1回あたり0.0001ドル払う「都度課金型API」
- ロボット・ドローン・自動運転車などの機械同士の自律決済

**なぜ知っておくべきか**
①**「AIエージェント経済」が技術的構想から実装段階に移った**シグナル。AI同士の自律的経済活動という新しい市場カテゴリの土台ができた、②USDCの利用拡大が**「人間の決済」から「機械の決済」へ用途を広げる**設計であり、ステーブルコイン市場の成長軸が増えた、③11チェーン対応という時点で**「特定チェーン専用ソリューション」ではなく業界標準のインフラ**を目指している。サークルのIPO[※21]・上場準備とも整合する戦略。

**ビジネスへの応用**
①**自社サービスのAPIに「都度課金（0.01ドル/回など）」のメニュー追加**が現実的になった。今までは決済手数料が見合わなかった単発呼び出しでも収益化できる、②**社内業務でAIエージェントを使う場合、エージェント同士の経費精算**の概念がいずれ必要になる。早めに「AIが消費するクラウド・API費用」を可視化する仕組みを整えておくとよい、③**B2B SaaSを運営している企業**は、自社が「AIエージェントに売られる側」「AIエージェントに買われる側」のどちらにもなる可能性を想定し、料金体系を超細分化する選択肢が出てきた。

スイスの資産運用会社**21シェアーズ（21Shares）**が組成した「**21shares Canton Network ETF**」（ティッカー: TCAN）が5月7日にナスダックに上場した。米国でカントンコイン（CC）現物ETFが上場するのは初。カントンコインは**機関投資家向け許可型ブロックチェーン「カントン・ネットワーク」**のネイティブトークンで、ナスダック・Visa・ゴールドマンサックスなど大手金融機関が参加するプライベートチェーン[※22]の決済単位。**月間トークン化資産取引高4兆ドル超**を支えるインフラ。**ビットコイン・イーサリアム以外の現物ETF**が米国に上場した点に意味があり、**「BTC・ETH→他アルトコイン[※23]へのETF拡張」**が次のフェーズ。21シェアーズは2025年11月にパリ・アムステルダムでもCC関連ETP[※24]を上場、5月6日には**ストラテジー優先株（STRC）連動ETN[※25]「21shares Strategy Yield ETN」**を英ロンドン証取に上場と、**仮想通貨派生商品の品揃えを世界の取引所に拡大中**。前回扱ったDTCCトークン化サービスの10月開始と組み合わせると、**「伝統金融商品の中に仮想通貨商品が混ざる」状況**が一気に普通になる流れ。

## 💡 今日の理解度チェック（3問）

**Q1.** ★入門 ETF（上場投資信託）の現物型と先物型はそれぞれどう違うか、1文ずつで説明してください。

**Q2.** ★★応用 日本国債のT+0レポ取引が実現すると、銀行の財務管理にどんな変化が起きると予想できますか？

**Q3.** ★★★発展 サークルのナノペイメンツのような「ガス代不要・極小額決済」インフラの普及は、ビジネスモデルにどんな新しい選択肢を生み出しますか？

<details>
<summary>答えを見る</summary>

**A1.** 現物型ETFは運用会社が実際の現物（例：ビットコイン）を保管し、その保有を裏付けにETFを発行する。先物型ETFは現物を持たず、先物市場の価格連動で値動きを作るためロールオーバーコストなど価格乖離が発生しやすい。需要が増えた時の現物買いインパクトがあるのは現物型。

**A2.** ①日中で取引を建てて当日中に解消することで、日締め時点のバランスシートに資産・負債が残らない状態を作れる、②自己資本比率規制・レバレッジ規制の負担軽減につながり、銀行は手持ち資本をより効率的に運用できる、③決済日数短縮で資金繰り計画の精度が上がる。一方、システム障害時の即時影響も大きくなるため、運用体制の高度化が求められる。

**A3.** ①従来「決済手数料が見合わない」として収益化を諦めていた単発・極小額のサービスが商業化可能になる（例：API呼び出し1回あたり0.001ドル課金）、②AIエージェント同士の自動取引市場が成立し、企業は「AIエージェントに買われる側」「売る側」両方の役割を担える、③従量課金が極限まで細分化できることで、「使った分だけ」の純粋な価値交換が可能になり、サブスクリプション型の固定費モデルとは異なる収益設計が生まれる。

</details>
