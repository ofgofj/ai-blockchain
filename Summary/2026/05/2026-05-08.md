## 🎓 今日の基礎講座：RWA（実資産トークン化）って何？

今日の記事の3割（オンド・DTCC・イーサリアム上の80億ドル米国債など9件）が**RWA**というテーマに集中した。これを理解しないと、今の業界の最大級のお金の流れが見えなくなる。

**RWA（Real World Asset、リアル・ワールド・アセット＝実物資産）**とは、株式・債券・不動産・金など現実世界に存在する資産を、ブロックチェーン上の「トークン」（＝デジタルな権利証）として表現する仕組みのこと。

**具体例で言うと:**
ある投資家が10万ドルの米国債を持っているとする。これを「トークン化」すると、その米国債を所有している証拠がブロックチェーン上に1つの「OUSG」というトークンとして記録される。投資家は、紙の証券を動かさなくても、トークンを送金するだけで権利を移転できる。

**何が便利になるか:**
- **24時間取引可能**：従来の株式・債券市場は平日昼間しか動かないが、ブロックチェーンは年中無休
- **数秒で決済完了**：今までは銀行間送金で数日かかっていたものが、5秒で終わる（オンドの実証実験）
- **小口化できる**：1億円のビルを「1万分の1」に分けて1万円から購入できる

**今までの障害は何だったか:**
ブロックチェーン上で資産処理は速くても、最終的に**現金を銀行から銀行へ動かす部分（法定通貨決済）**は伝統的な銀行ネットワークに頼るしかなく、ここで時間とコストがかかっていた。今日のオンド連合の実験は、**この「ブロックチェーンと銀行をつなぐ最後の一マイル」を埋める**取り組み。

今日の記事では、このRWAに関するニュースが**5件以上**（うち詳細3件）あります。

## 決済・送金

### ★★★ オンド・JPモルガン・マスターカード・リップル4社連合、トークン化米国債を5秒で国際銀行間償還
**ソース:** [オンド、JPモルガンやリップルらとトークン化米国債の国際銀行間償還を試験運用](https://www.neweconomy.jp/posts/571458)（あたらしい経済）／[リップル×JPモルガンが示したRWAの未来、トークン化米国債を5秒で償還](https://crypto-times.jp/news-the-future-of-rwa-as-revealed-by-ripple-and-jp-morgan/)（CRYPTO TIMES）

**ひとことで言うと** — トークン化米国債の国境・銀行をまたぐ償還を、4社のシステムをつないでほぼリアルタイム（5秒未満）で完了させる実証実験に成功した。

**何が起きたか**

オンドファイナンス（Ondo Finance）が5月6日、JPモルガンの企業向け決済基盤キネクシス（Kinexys by J.P. Morgan）、マスターカード（Mastercard）、リップル（Ripple）と共同で、トークン化米国債ファンドのパイロット取引を実施したと発表。

| 工程 | 担当 | 内容 |
|---|---|---|
| ①トークン償還 | リップル/オンド | XRPL[※5]上で「OUSG」（オンド短期米国債トークン）を償還処理（5秒未満で完了） |
| ②指図ルーティング | マスターカード | MTN（マルチトークン・ネットワーク）で銀行ネットワークに払い出し指図 |
| ③法定通貨送金 | JPモルガン | コルレス銀行ネットワーク経由でリップルのシンガポール口座に米ドル送金 |

つまり**「ブロックチェーン上の資産処理 → 銀行間ネットワーク上の法定通貨決済」を1本のフローでつないだ**世界初の事例。

**なぜ知っておくべきか**
①従来、トークン化米国債は「ブロックチェーンの中だけなら速く動くが、現金化（償還）すると数日かかる」という最大の課題があった。今回の実証で**「銀行営業時間の壁」「電信送金の手作業」**を突破する道筋が示された、②**米国大手金融4社が一緒に動いた**点が重要。実証実験の段階から、ブラックロックやフィデリティといった本命勢が控える本格展開フェーズに移行する前段階、③イーサリアム上のトークン化米国債は同時期に**80億ドルを突破（半年で倍増）**しており、需要側のインフラも整いつつある。

**ビジネスへの応用**
中小企業の経営者・財務担当者が押さえる3点。①**自社の海外取引先への決済**（特にシンガポール・香港・米国）でいずれ「24時間オンチェーン決済」のメニューが出てくる可能性が高い。送金日数短縮は資金繰りに直結するため、メインバンクに「トークン化決済の取扱予定」を聞いておくとよい。②**自社の余剰資金運用**でトークン化米国債（OUSG等）が選択肢になる。最低取引額が下がれば、中小企業の短期資金運用先として機関投資家商品が手の届く範囲に入る。③**取引所・証券会社が伝統金融側と連携する流れ**は今後1〜2年加速する。自社のメインの証券口座運用先が「トークン化対応」を表明したかは、来年の決算発表時に確認する習慣をつけたい。

### ★★★ DTCC（米証券インフラ最大手、114兆ドル保管）、トークン化サービスを10月に正式開始
**ソース:** [DTCC、証券トークン化サービスを10月に開始へ。7月に限定本番取引を予定](https://www.neweconomy.jp/posts/571464)（あたらしい経済）

**ひとことで言うと** — 米国の株や債券のあらゆる売買の「決済・保管」を裏で担う**DTCC**（米国の証券インフラの心臓部）が、株式・ETF・米国債をブロックチェーン上のトークンとして扱うサービスを2026年10月に開始する。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| 7月 | 限定的な本番取引を開始 |
| 10月 | サービス正式開始 |
| 対象資産 | ラッセル1000指数構成銘柄[※6]、主要ETF、米国債（短期・中期） |
| 監督下 | 米SECからノーアクションレター[※7]取得済 |
| 保管資産 | 114兆ドル（約1京7,943兆円）超 |
| 参加企業 | ブラックロック、ゴールドマン・サックス、JPモルガン、ナスダック、NYSE、サークル、ロビンフッドなど50社以上 |

**重要な設計思想:**
今回のサービスは「株を新規にブロックチェーンに移す」のではない。**DTCで保管されている既存証券の権利情報を、トークンとして記録・移転する**仕組み。配当権・議決権・所有権はすべて従来通り。NYSEとナスダックも今年中にトークン化銘柄を既存の注文板で取引できる規則変更を承認済で、**DTCC稼働を待って一斉始動**の構造。

**なぜ知っておくべきか**
①**米証券市場の「土台」がブロックチェーン化する**転換点。DTCCは米国の株式取引の100%・債券取引のほぼすべての清算決済を担っているため、ここがトークン化対応すれば、伝統金融とブロックチェーンの境界線が事実上消える、②前述のオンド連合のような「公的なパブリックチェーン上での実証」と、DTCCのような「クローズドな金融機関ネットワーク内のトークン化」**の二本立てで進行**する構図が見えてきた、③参加企業リストにブラックロック・JPモルガン・ナスダック・NYSEがそろっている時点で、**規制当局が「これは止めない」と決めたシグナル**として読み取れる。

**ビジネスへの応用**
①**自社の株式関連業務**（IR、株主管理、配当処理）で、いずれ「トークン化銘柄」の処理パターンが標準業務に入ってくる。来年以降の新人教育カリキュラムに含めることを検討する価値あり、②**B2B資金調達手段としての社債発行**で、将来的に「トークン化社債」が選択肢に入る可能性が高まる。発行コストが下がれば中小企業のファイナンス手段が広がる、③日本のJPX（東証）も**早ければ2027年に暗号資産ETF上場の方針**を表明（同日報道）。米国の動きが日本に1〜2年遅れで波及するパターンが定着しているため、**「米国DTCC10月→日本2027〜2028年」**のロードマップで自社対応を逆算するとよい。

### ★★★ ストラテジー社、Q1赤字125億ドル｜セイラー氏「BTC売却の可能性」明言で永久保有戦略から方針転換
**ソース:** [ストラテジー社125億ドルの巨額赤字、BTC含み損が直撃](https://crypto-times.jp/news-strategy-inc-suffers-massive-12-5-billion-loss/)（CRYPTO TIMES）／[「絶対売らない」セイラー氏が変節？BTC売却の可能性を明言](https://crypto-times.jp/news-mr-sayler-confirmed-the-possibility-of-selling-his-btc/)（CRYPTO TIMES）／[ストラテジー社のBTC戦略に異変？優先株「STRC」がトークン化か](https://crypto-times.jp/news-is-there-something-unusual-going-on-with-strategy-inc-s-btc-strategy/)（CRYPTO TIMES）

**ひとことで言うと** — 世界最大のBTC保有上場企業ストラテジー（旧マイクロストラテジー）が、**BTC価格下落で約125億ドルの赤字**を計上した一方、創業者セイラー氏が「**配当支払いのためにBTCを売る可能性がある**」と発言し、長年の「絶対売らない」ナラティブから方針転換を示した。

**何が起きたか**

| 項目 | 内容 |
|---|---|
| Q1純損失 | 約125億4,000万ドル |
| 未実現損失 | 144億6,000万ドル（BTC価格下落起因） |
| 保有BTC | 818,334枚 |
| 平均取得価格 | 約75,537ドル/BTC |
| デジタル資産評価額 | 約618億ドル（5/3時点）→約668億ドル（直近） |
| 年初来BTC調達額 | 116億8,000万ドル（2025年同期比で加速） |
| 新方針 | STRC優先株配当をBTC売却益で支払うシナリオを公式化 |

**セイラー氏の発言の3つの要点:**
①「**負債でBTCを買い、値上げ後に一部売却して配当を支払う**」というキャッシュフロー設計を初めて明言、②「**100万BTC到達は時間の問題**」と保有量拡大は継続、③「市場を**インオキュレート（免疫化）**するために一部売却する」つまり「いざとなれば売れる」事実を市場に示すことで信用を強化する戦略。

**STRCのトークン化計画:**
同社CEOフォン・レー氏は、米CLARITY法成立を前提に**STRC優先株を基盤としたレイヤー2[※8]インフラ「Apyx」の開発加速**を表明。配当を提供しつつBTC原資産にリンクする金融商品をブロックチェーン化する構想で、**伝統金融商品×BTCのハイブリッド型**をさらに進化させる動き。

**なぜ知っておくべきか**
①BTC財務戦略を採用した上場企業（メタプラネット、テスラ等）が**実損計上局面に直面**する初期事例。会計上の含み損が即赤字になる**フェアバリュー会計**の影響が今後も他社で連鎖、②セイラー氏の方針転換は**「BTCは絶対売らない」というシンプルな信仰モデル**から、**「BTCを資産運用対象として保有・売却する」プロモデル**への移行を意味する。これは機関投資家の参加を促進する一方、個人投資家の「永久保有HODL」ナラティブを揺さぶる、③同社のBTC平均取得価格（75,537ドル）から見ると、現在価格8万ドル台はかろうじてプラス圏。**8万ドル割れすると含み損→さらなる赤字計上の連鎖**が起きるため、メタプラネットなど他社の同期決算にも波及する。

**ビジネスへの応用**
①**自社のIR説明資料に「将来的なBTC財務採用の可能性」を盛り込んでいる場合、ストラテジーの実損事例を必ず参照**する必要が出てきた。「永久保有」の前提が崩れた以上、計画は再設計が必要、②**配当付き優先株式（STRC）×BTC＋トークン化**というハイブリッド型金融商品設計は、**中小企業の資金調達でも応用が広がる**。BTC含有量・固定配当・流動性確保の3要素のバランスをどう設計するかが新しい財務テーマに、③**自社で取引先からBTCで決済を受ける場合、四半期毎の評価損益の計上ルール**を経理担当が事前に把握すべき段階に。日本基準では時価評価が原則、米国基準でもフェアバリュー会計に統一済。

## 💡 今日の理解度チェック（3問）

**Q1.** ★入門 RWAトークン化とは何か、専門用語を使わず1文で説明してください。

**Q2.** ★★応用 DTCCのトークン化サービスが10月に始まると、米国の株式取引にどんな実務的変化が起きると予想できますか？

**Q3.** ★★★発展 ストラテジー社のセイラー氏の発言（「BTC売却もありうる」）は、機関投資家にとってのBTC評価をどう変えると考えられますか？

<details>
<summary>答えを見る</summary>

**A1.** 株や米国債のような現実世界の資産を、ブロックチェーン上のデジタルな権利証（トークン）として表現する仕組み。これにより24時間取引・数秒決済・小口化が可能になる。

**A2.** ①取引時間の延長または24時間化（ナスダック・NYSEのトークン化注文板始動）、②決済時間の短縮（現在のT+1（取引日翌日）からほぼリアルタイム化）、③ブロックチェーン上での裏付けによる透明性向上。ただし配当・議決権など権利関係は従来と同じに維持されるため、投資家側の体感的な「お金の動きが速くなる」が主な変化となる。

**A3.** プラスの側面とマイナスの側面の両方がある。プラスは「いざとなれば売却できる」流動性が確認されたことで、機関投資家がBTCを「保有不能リスクのある投機資産」から「アセットマネジメント対象の運用商品」として扱いやすくなる点。マイナスは「絶対売らない」前提で形成されてきた個人投資家のHODL[※18]文化が動揺し、価格下落時に「ストラテジーが売るかも」という観測売りが連鎖する可能性が高まった点。長期的にはプロ投資家の参入を促進する一方、短期的には価格ボラティリティを増幅する効果がある。

</details>
